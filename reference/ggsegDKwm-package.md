# ggsegDKwm: Desikan-Killiany White-Matter Parcellation Atlas for the 'ggsegverse' Ecosystem

Axial Desikan-Killiany white-matter parcellation ('wmparc') together
with basal-ganglia structures (caudate, putamen, pallidum, thalamus),
provided as a 'ggseg_atlas' object for use with 'ggseg'. Cortical grey
matter is merged into a single background label. Built in symmetric
'fsaverage_sym' template space with axial slice views.
